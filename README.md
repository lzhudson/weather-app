It is a simple application that allows you to access [Open Weather](https://openweathermap.org/api).

## Running Project
1. Clone Repository:
```bash 
git clone https://github.com/lzhudson/weather-app.git
```
2. Enter in folder:
```bash
cd weather-app
```
3. Install packages:
```bash
yarn 
```
4. Running in mode development:
```bash
yarn dev
```
